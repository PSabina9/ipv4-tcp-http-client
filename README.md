# ipv4-tcp-http-client
TCP/IPv4 HTTP client in C for downloading default web pages on Linux.


**Course:** Telecommunications Networks  
**Year:** 2025  

**Overview**

This project implements a simple **HTTP client in C** that runs on a workstation terminal (tested using **PuTTY**). The client uses **IPv4 and TCP** protocols to connect to a chosen web server and download its default page. It demonstrates basic network programming concepts: socket creation, TCP connection management, sending HTTP requests, and saving server responses.  

**Features**

- Connect to a web server using TCP/IPv4  
- Send HTTP GET requests  
- Receive server responses  
- Save the downloaded page locally (`index.html`)  
- Interactive menu to connect, disconnect, send requests, or exit
